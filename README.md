# 🏥 Sistema de Teleagendamento Médico (Java CLI)

## 📌 Sobre o Projeto

Este projeto consiste em um sistema de teleagendamento médico desenvolvido em **Java**, executado via terminal (CLI). O objetivo é simular um ambiente real de agendamento de consultas, aplicando conceitos fundamentais de desenvolvimento back-end.

O sistema permite gerenciar consultas médicas de forma simples e organizada, incluindo cadastro, visualização, atualização e remoção de agendamentos.

---

## 🚀 Funcionalidades

* ✅ Criar novo agendamento
* 📋 Listar todos os agendamentos
* 🔍 Buscar agendamento por ID
* ✏️ Atualizar/Reagendar consulta
* ❌ Cancelar agendamento

---

## 🧠 Conceitos Aplicados

* Programação Orientada a Objetos (POO)
* Estrutura de dados com `ArrayList`
* Separação de responsabilidades (Model / Repository / Main)
* CRUD completo (Create, Read, Update, Delete)
* Interação com usuário via terminal

---

## 🛠️ Tecnologias Utilizadas
